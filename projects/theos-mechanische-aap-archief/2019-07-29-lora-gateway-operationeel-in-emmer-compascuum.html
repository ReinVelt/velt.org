Lora gateway operationeel in Emmer-Compascuum

2019-07-29 - Theo's Mechanische Aap

Vanaf aankomend weekend is de Lora gateway bij de vestiging van Theo's Mechanische Aap in Emmer-Compascuum operationeel. Het gaat hier om een multichannel gateway van het type Lorank8. De gateway werkt met de 868Mhz band. De Lora gateway is in principe voor iedereen gratis te gebruiken. Alle data die op de gateway binnen komt wordt doorgestuurd naar TTN (The Things Network) en Semtech. Op verzoek kan data ook naar andere servers worden doorgestuurd. Neem doorvoor even contact met ons op.Naas t de Lora gateway (868Mhz) is er afgelopen week ook een gateway voor de 433 Mhz band opgetuigd.
